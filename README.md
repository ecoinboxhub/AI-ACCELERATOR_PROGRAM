# AI Accelerator Program

This repository contains learning materials (slides, labs/code), datasets, and project templates for participants.

## Repo structure

```
AI-ACCELERATOR_PROGRAM/
  01_curriculum/            # Syllabus, schedule, module outlines
    modules/                # Module-by-module outlines
    weeks/                  # Week-by-week plan (optional)
  02_slides/                # Slide decks per module
    module_00/
    module_01/
    module_02/
  03_labs/                  # Hands-on labs (code + instructions)
    lab_00_setup/
    lab_01_basics/
    lab_02_ml/
    lab_03_llm/
  04_projects/              # Capstone projects and rubrics
  05_datasets/              # Datasets used in labs/projects
    raw/                    # Original data (often large)
    processed/              # Cleaned/feature-ready data
    external/               # Third-party data (store links/license notes)
    sample/                 # Small sample data committed to git
  06_readings/              # PDFs/links/notes for optional reading
  07_tools/                 # Utility scripts (downloaders, validators, etc.)
  08_templates/             # Templates: reports, notebooks, project skeletons
  .github/                  # (Optional) Issues/CI for managing updates
```

## How participants should use this repo

- Start with `01_curriculum/` to see the schedule and what to do each week.
- Slide decks live in `02_slides/`.
- Lab instructions and code live in `03_labs/`.
- Datasets live in `05_datasets/` (see notes inside).
